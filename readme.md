# Spolupráca s Hory Doly

[Hory Doly](https://www.horydolylm.sk/) (HD) je kultúrno-komunitný priestor v Liptovskom Mikuláši. Vyplnil som [formulár pre organizátora](https://www.horydolylm.sk/chcem-podujatie/) a spolupráca postupne naberala kontúry.

Všetky materiály, ktoré v HD budem používať, umiestnim sem, aby boli pre kohokoľvek kedykoľvek dostupné online. [Na GitHube](https://github.com/tiborepcek/hory-doly) mi môžete navrhnúť úpravy alebo si vytvorte fork.
